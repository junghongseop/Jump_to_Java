-클래스 명-
자바 프로그램은 클래스 단위이기 때문에 자바 프로그램을 만드는 것은 자바 클래스를 만드는 것과도 같음
자바 클래스를 만들려면 자바 클래스에 대한 이름을 지어야 함

<규칙>
-클래스 명은 명사로 함
-여러개의 단어가 섞이는 경우 각 단어의 첫 번째 문자는 대문자이어야 함

-메서드 명-
-메서드 명은 동사로 함
-클래스명과 마찬가지로 여러개의 단어가 섞이는 경우 각 단어의 첫번째 문자는 대문자이어야 함
 단, 처음 시작하는 문자는 항상 소문자

-변수명-
-변수 이름은 짧지만 의미가 있어야 함
-순서를 의미하는 임시적인 정수의 변수명은 i,j,k,m,n을 사용
-변수명에 _, $기호를 사용할 수 있지만 시작 문자로 사용하지 않음
